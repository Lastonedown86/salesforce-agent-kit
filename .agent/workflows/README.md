# Salesforce Developer Workflows

Step-by-step guidance for common Salesforce development tasks. Use these workflows to guide you through complex implementations.

## How to Use

Reference a workflow by name when asking for help:
- "Follow the apex-development workflow to create a batch job"
- "Use the lwc-development workflow to build a component"
